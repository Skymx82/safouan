2. Architecture MVC

Principe de fonctionnement

  1. L'utilisateur accède à index.php avec un paramètre action
  2. index.php (Front Controller) route la requête vers le contrôleur approprié
  3. Le contrôleur appelle le modèle pour récupérer les données
  4. Le modèle interroge l'API REST via des requêtes HTTP
  5. L'API REST accède à la base de données MySQL via PDO
  6. Les données sont renvoyées en JSON au modèle
  7. Le contrôleur charge la vue appropriée avec les données
  8. La vue génère le HTML affiché à l'utilisateur (avec Bootstrap)
